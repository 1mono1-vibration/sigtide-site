Why daily-only, and how to read this honestly

Other sites publish 15-minute and hourly RSI. We deliberately don't: this page regenerates once per day with the daily scan, so an intraday number here would be stale within hours — and a stale number presented as live is worse than no number. Daily RSI is the timeframe our scoring, backtests and regime work actually run on, so everything on this page is measured the same way, on the same bars, every day.
